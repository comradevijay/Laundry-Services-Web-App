# 🧺 Laundry Services Web Application

A modern, responsive **Laundry Service Web Application** that allows users to browse laundry services, select items, view total pricing, and submit booking details through an interactive interface. This project demonstrates strong fundamentals in frontend development, UI design, and client‑side logic implementation.

## 📌 Project Objective

The goal of this project is to build a lightweight, user‑friendly laundry booking interface using core web technologies without relying on external frameworks. It highlights practical understanding of DOM manipulation, event handling, responsive layouts, and clean UI structuring.

---

## ✨ Key Features

* Interactive service selection system
* Dynamic cart with real‑time total calculation
* Booking form with user input fields
* Responsive design for mobile and desktop
* Clean and intuitive user interface
* Lightweight and fast loading (no frameworks used)

## ⚙️ Installation & Setup

To run this project locally:

```bash
git clone https://github.com/comradevijay/Laundry-Services-Web-App.git
cd Laundry-Services-Web-App
```

Then open:

```
index.html
```

in any modern browser.

No server or additional dependencies are required.

---

## 🧠 Functional Workflow

1. Users browse available laundry services listed with pricing.
2. Selected services are added to a dynamic cart.
3. The application automatically calculates totals.
4. Users submit booking details via the form.
5. Submission interaction is handled on the client side.
